Vibrant Inventory Project

InventoryDashboard

About

This is a colorful, static Inventory Management System with a dashboard. It stores data locally in the browser using localStorage. It's perfect for showcasing on GitHub Pages or as a student project.

Deploy

  1. Create a GitHub repo and upload these files.
  2. Enable GitHub Pages (branch: main / root folder).
  3. Visit index.html to manage inventory and dashboard.html for charts.

Features

  • CRUD operations for items (name, category, qty, price, description)
  • Search and filter
  • Import/export JSON, clear all data
  • Dashboard with Chart.js visualizations
